dtree 数据树文档 - layui.dtree
以下介绍了在实际开发中树的基本使用方法，也是比较常见的使用。
模块加载名称：dtree
点击任意节点回调节点数据

使用树时，你肯定需要用到这个树中所带的数据，那么只需要按照这种方式写，数据自然会出现在你的眼前。

    									// 渲染，使用data渲染
									dtree.render({
									  elem: "#commonTree1",
									  data: commonTree1
									});
									
									// 点击节点触发回调
									dtree.on("node('commonTree1')" ,function(param){
									  layer.msg(JSON.stringify(param));
									});
